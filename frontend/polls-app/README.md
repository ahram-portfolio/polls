# 투표 Frontend (with. Angular)

## 실행방법
1. 터미널에서 `npm install` 을 해주세요.
2. 터미널에서 `ng serve`를 입력해주세요.
3. `localhost: 4200` 으로 접속해서 확인하시면 됩니다. (크롬 브라우저 권장)

## Unit Test 실행방법
1. 터미널에서 `npm install` 을 해주세요.
2. 터미널에서 `ng test`를 입력해주세요.
